# Health & Fitness Tracker

## How to Run the App

1. **Clone this repository**
2. **Install dependencies:**
   ```
   npm install
   ```
3. **Set up the database (optional):**
   - Run `create_db.sql` and `insert_test_data.sql` if you want to use MySQL (not required for marking)
4. **Start the app:**
   ```
   node index.js
   ```
5. **Open your browser:**
   - Go to [http://localhost:8000](http://localhost:8000)

## Features

- **Login & Session:** Secure login with a pre-created user (see above)
- **Profile:** View and update your personal info (name, email, height, weight, etc.)
- **Achievements:** Log and view your fitness achievements directly from the Achievements page
- **Fitness Level:** See your progress from Beginner to Advanced as you log more activities
- **My Plans:** Choose a goal (stay healthy, lose weight, gain weight) and get a tailored fitness/diet plan
- **Activity History:** See a timeline of all your logged achievements
- **Settings:** Update your details anytime
- **Modern UI:** Clean, accessible design with a purple/grey color scheme
